# SQL Sales Analysis — Chinook Database

## Описание проекта
Проект посвящён анализу продаж цифрового музыкального сервиса на основе учебной базы данных Chinook.  
В рамках проекта были рассчитаны базовые бизнес- и продуктовые метрики, исследована динамика выручки, проведён анализ продаж по странам, а также выполнена сегментация клиентов по количеству заказов.

## Цель проекта
Показать навыки работы с SQL при анализе бизнес-данных:
- расчёт ключевых метрик
- агрегация и группировка данных
- анализ динамики
- сравнение стран и клиентов
- сегментация пользователей

## Используемые данные
В проекте использована база данных Chinook, моделирующая цифровой музыкальный магазин.

Ключевые таблицы:
- `customer` — клиенты сервиса
- `invoice` — заказы
- `invoice_line` — состав заказов
- `track` — треки
- `album` — альбомы
- `artist` — исполнители

## Что было сделано
В SQL были рассчитаны следующие показатели:
- общее число клиентов
- общее число заказов
- общая выручка
- средний, минимальный и максимальный чек
- выручка по дням
- количество заказов по дням
- средний чек по дням
- число активных клиентов по странам
- количество заказов по странам
- выручка по странам
- средний чек по странам
- топ клиентов по выручке
- топ клиентов по числу заказов
- ARPU
- сегментация клиентов по числу заказов

## Основные результаты
По итогам анализа были получены следующие общие показатели:
- количество клиентов: 59
- количество заказов: 412
- общая выручка: 2328.60
- средний чек: 5.65
- ARPU: 39.47

Географический анализ показал, что:
- наибольшую выручку приносит USA
- по числу заказов также лидирует USA
- далее по выручке выделяются Canada, France и Brazil

Сегментация клиентов по числу заказов показала, что почти все клиенты относятся к группе `6+ order group`, что говорит о высокой однородности клиентской базы в рамках данного датасета.
